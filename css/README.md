# CSS Folder

Purpose:
- Contains the full visual system for the public site and admin panel.

Files:
- variables.css: Global design tokens and palette variables.
- themes.css: Light and dark theme token mapping.
- base.css: Resets, typography, and base element rules.
- layout.css: Grid, containers, responsive structure.
- components.css: Reusable UI component styles.
- sections.css: Section-specific styles.
- animations.css: Loading and motion definitions.
- admin.css: Admin panel styling.

Interaction:
- index.html and admin.html import these files in cascade order to keep style concerns modular.
